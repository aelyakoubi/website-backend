# Development Setup

## Install Dependencies

```bash
npm install
```

---

# Prisma Setup

## Initialize Prisma

```bash
npx prisma init
```

## Generate Prisma Client

```bash
npx prisma generate
```

## Format Prisma Schema

```bash
npx prisma format
```

## Push Schema to Database

```bash
npx prisma db push
```

## Create Migration

```bash
npx prisma migrate dev --name init
```

---

# Seed Database

```bash
npm run seed
```

---

# Start Development Server

```bash
npm run dev
```

---

# Open Prisma Studio

```bash
npx prisma studio
```

Prisma Studio opens a browser interface to inspect and manage the database.

---

# Reset Database

```bash
npx prisma db push --force-reset
npm run seed
```

This will:

* remove existing database data
* apply the latest Prisma schema
* reseed the database

---

# Useful Prisma Commands

## View all Prisma commands

```bash
npx prisma --help
```

## Create a new migration

```bash
npx prisma migrate dev --name migration_name
```

## Regenerate Prisma Client

